HTML Tags

HTML consists of various tags that define the structure and content of a web page. Some common tags include:

  • <h1> - <h6>: Heading tags
  • <p>: Paragraph tag
  • <a>: Anchor tag for hyperlinks
  • <img>: Image tag
  • <ul> and <li>: Unordered list and list item tags
  • <div>: Division tag for grouping elements

Styling

You can also add CSS styles to HTML elements to change their appearance. Styles can be defined inline using the style attribute or in an external CSS file.

JavaScript

HTML can also incorporate JavaScript for dynamic behavior and interactivity.
